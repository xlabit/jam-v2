### 🧩 Shared Design Rules

* **Card structure:**

  ```html

    <div class="cta-content w-full md:w-1/2">

      <p class="text-base opacity-90 mb-4">Subheadline</p>
      <a href="..." class="btn-primary">Button</a>
    </div>
    <div class="cta-image w-full md:w-1/2">
      <img src="..." alt="..." loading="lazy" class="rounded-lg" />
    </div>
  </div>
  ```

* **Animations:**

  * Fade-in upward (`opacity: 0 → 1`, `transform: translateY(20px) → 0`)
  * Duration: 0.5s per card, 0.1s staggered delay.

* **Accessibility:**

  * Use `<section>` + `<article>` + `<h3>` structure.
  * Add `aria-label` and descriptive alt text.
  * Keyboard navigable buttons.
  * Ensure color contrast meets **WCAG 2.1 AA**.
